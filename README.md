# Aircraft Component Data Model

A Python-based data modeling framework for aerospace engineering applications.

Production-ready Python data modeling framework for aerospace engineering with REST API and database integration.

## Features
- ✅ Object-Oriented Design with dataclasses
- ✅ Multi-format serialization (JSON, YAML, HDF5)
- ✅ SQLAlchemy ORM with PostgreSQL/SQLite support
- ✅ FastAPI REST endpoints with auto-generated docs
- ✅ Comprehensive pytest test suite (>90% coverage)
- ✅ Schema validation with Pydantic
- ✅ Version management with automatic backups
- ✅ Cross-platform compatibility (Windows/Linux)

## Quick Start
```bash
pip install -r requirements.txt
pytest tests/ -v          # Run tests
python run_api.py         # Start API server
